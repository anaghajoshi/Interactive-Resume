Embedded Systems (Hardware + Software)

RFID Based Electronic Road Pricing (ERP) System

Radio-frequency identification (RFID) is an automatic identification method wherein the data stored on RFID tags or transponders is remotely retrieved. The RFID tag is a device that can be attached to or incorporated into a product for identification and tracking using radio waves. Some tags can be read from several meters away, beyond the line of sight of the reader. RFID technology is used for a smarter toll collection system called “Electronic Road Pricing”. The system normally consists of an objects counter, sensors, display board, the controllable gate, RFID tags and RFID reader. Using this hardware with compatible VB software and embedded c programming a working prototype was developed with AT89C52 as the microcontroller. The tag was placed on cars and the reader at the toll plaza to automate the toll collection.

Parallel Programming (Hybrid MPI + CUDA)

Accelerated Gaussian Elimination using MPI CUDA Hybrid (pivoting, pipelining, LU Decomposition)

One of the most popular techniques for solving simultaneous linear equations is the Gaussian elimination method. This project presents a comparative analysis of different techniques for Gaussian Elimination method via three different algorithms- partial pivoting, parallel-pipelined and Lower-Upper (LU) decomposition. Additionally, each of these three algorithms are implemented in four different ways – serial C code, Parallel MPI code, Parallel CUDA code and parallel Hybrid MPI+CUDA code. The hybrid implementation is the very first to the best of our knowledge. Hybrid code is aimed at leveraging the CPU+GPU heterogeneous Environment. The performance results for these twelve codes are key in gaining the in-depth insights about the compatibility among three constituents – application algorithm, Language of code and Hardware Platform, of a usual High Performance Computing (HPC) eco-system.
